# Security and Privacy

University Software Copyright Helper 面向软著材料整理，使用时可能接触项目源码、截图、联系人、证件号码和单位信息。请在开源、提 issue 或提交 PR 前做好脱敏。

## 不要提交的信息

- 真实身份证号、统一社会信用代码、手机号、邮箱、地址等身份信息。
- 未公开的软件源码、截图、系统账号、接口地址、密钥或测试数据。
- 完整申请表、说明书、源代码鉴别材料、合同、协议或学校内部文件。
- 包含个人或团队敏感信息的 Word、PDF、压缩包或图片。

## 推荐做法

- 使用虚构项目名、虚构联系人和脱敏后的单位信息提交示例。
- 截图只保留界面结构，模糊或替换真实姓名、账号、数据和路径。
- 示例代码和提示词只展示字段结构，不展示真实权利归属和证件号码。
- 在公开 issue 中描述问题现象，不粘贴完整真实材料。

## 报告安全或隐私问题

如果你发现仓库中存在敏感信息、错误的隐私处理建议或可能导致泄露的流程，请优先通过私有渠道联系维护者；如果只能使用 GitHub issue，请不要粘贴敏感原文，只说明受影响文件和问题类型。

## 责任边界

本项目不保存用户数据，也不提供云服务。使用者在本地运行 Codex 或其他 AI 工具时，应自行确认所用工具、模型和平台的数据处理政策是否符合所在单位要求。
